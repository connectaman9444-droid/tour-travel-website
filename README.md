# Expense Tracker

## Description
Expense Tracker is a simple web application that helps users manage their daily finances. 
Users can add income and expenses, track their spending, and see their current balance.

This project was built to practice basic web development concepts.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Add income and expense transactions
- View total balance
- Simple and user-friendly interface
- Track daily spending

## How to Use
1. Open the application in your browser.
2. Add your income or expense.
3. The app will automatically update your balance.

## Project Purpose
This project was created as a beginner web development project to improve JavaScript and front-end development skills.
